=====================================================
 تأكيد الحضور (RSVP)
=====================================================
الزر «أؤكّد حضوري» يفتح نافذة، الضيف يكتب اسمه وعدد الأفراد،
وأي تأكيد يُكتب فورًا في الملف الجاهز:  rsvp.json

التشغيل:
  1) من مجلد المشروع:  node server.js
  2) افتح في المتصفح:  http://localhost:3000
  3) كل تأكيد حضور يظهر مباشرة كعنصر جديد داخل rsvp.json
     مثال:
       [
         { "name": "محمد", "count": 2, "date": "..." }
       ]

ملاحظة: لازم السيرفر يكون شغّال (node server.js) عشان البيانات تتكتب في الملف.
